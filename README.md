# To-Do List Application

## 📌 Description
This is a simple Python-based To-Do List application that helps users manage daily tasks by adding, viewing, and deleting tasks.

## 🚀 Features
- Add tasks
- View tasks
- Delete tasks
- Workload classification (Light, Moderate, Heavy)

## 🧠 Concepts Used
- Lists
- Loops
- Functions
- Conditional statements

## ▶️ How to Run
1. Install Python
2. Run the file:
   python todo_list.py
